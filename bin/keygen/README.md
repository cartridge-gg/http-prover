
# Prover Key Generation

This is a simple command-line tool for generating new keys using the Prover SDK.

## Installation
1. Clone the repository:

```bash
git clone https://github.com/cartridge-gg/http-prover.git
```

## Usage
```bash
cargo run -p keygen
```
